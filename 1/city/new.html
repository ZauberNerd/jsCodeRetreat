Why?

There is no JavaScript Code Retreat in your city yet, but you want to have one!

How you get your city listed here?

We simply used github for the page you are looking at. And that's also how you can add your city or even contribute site fixes for this site.
Just follow these steps and you get on to this page here:

  • Fork this repo
  • Create a city page name it like de-munich.html (see here for more examples)
  • Create a pull request of your forked project
  • We pull, your site is up

How to host?

Get a room, some food and a projector in there. Tell the people and off you go. If you need some help on hosting there are some really good info at hosting and facilitating.
If you have specific question you can always post here.
